# RevPilot Gen-2 

## Key Features

* **Dataset Branching:** Experiment safely with Git-like version control for structured JSON and CV datasets.
* **RevPilot AI:** An algorithmic co-pilot capable of writing optimized C++, Java, and Python scripts on the fly.
* **Neural Workspace:** Built-in tools for data autocomplete and schema validation.
* **High-Speed Sync:** Optimized binary deltas for pushing and pulling massive data files without wasting bandwidth.

---

## 📂 Project Structure

This repository is split into two main sections:

* **`/frontend`**: Built with React and Tailwind CSS. Features a sleek, dark-themed UI with glassmorphism effects.
* **`/backend`**: Node.js/Express server handling authentication (Google & OTP), database connections, and AI agent integration.

---

## 🛠️ Tech Stack

**Frontend:**
- React (Vite)
- Lucide React (Icons)
- Framer Motion (Animations)
- Zustand (State Management)

**Backend:**
- Node.js & Express
- MongoDB (Data Persistence)
- Google OAuth 2.0
- Gemini 1.5 Pro (AI Engine)
